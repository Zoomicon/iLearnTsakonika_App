# iLearnTsakonika
iLearnTsakonika language learning app

Get in touch with the Tsakonian dialect (Tsakonika) of the Greek language.
Learn Tsakonika through interactive comics with audio dialogues and learning excercises for vocabulary and grammar.

[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE.txt)
[![Engine](https://img.shields.io/badge/Uses-READCOM.Core-blue)](https://github.com/Zoomicon/READCOM.Core.Delphi)

## Supported platforms
* [Windows](https://bit.ly/tsakonika-windows) 7+
* [Android](https://bit.ly/tsakonika-android) 5.0+
* iOS 10+
* MacOS-X

Also see [Releases](https://github.com/Zoomicon/iLearnTsakonika_App/releases) page
